# ระบบนัดหมายหมอ - คู่มือการใช้งาน

## 📋 ภาพรวมระบบ

ระบบนัดหมายหมอเป็นเครื่องมือจัดการการนัดหมายระหว่างหมอและผู้สูงอายุอย่างมีประสิทธิภาพ พร้อมการแจ้งเตือน SMS อัตโนมัติ

## 🎯 คุณสมบัติหลัก

### 1️⃣ สำหรับหมอ (Doctor)
- ✅ **สร้างนัดหมาย** - กำหนดวันเวลาและหมายเหตุให้ผู้สูงอายุ
- ✅ **ดูรายการนัด** - ตรวจสอบการนัดที่จะมา
- ✅ **ทำให้เสร็จ** - บันทึกการนัดเมื่อเสร็จสิ้น
- ✅ **เลื่อนวัน** - ปรับวันนัดหากไม่ว่าง พร้อมสาเหตุ
- ✅ **ยกเลิก** - ยกเลิกการนัดหากจำเป็น

### 2️⃣ สำหรับผู้สูงอายุ (Elderly)
- ✅ **ดูการนัด** - ตรวจสอบวันนัดอย่างชัดเจน
- ✅ **แจ้งเตือน SMS** - รับข้อความ SMS ล่วงหน้า 3 วัน
- ✅ **แจ้งเตือนวันนัด** - รับข้อความเตือนในวันนัด
- ✅ **เลื่อนวัน** - ขอเลื่อนวันได้
- ✅ **ยกเลิก** - ยกเลิกการนัดได้
- ✅ **คุยกับหมอ** - ส่งข้อความถาม เกี่ยวกับการนัด
- ✅ **ประวัติการนัด** - ดูประวัติการนัดที่ผ่านมา

### 3️⃣ สำหรับแอดมิน (Admin)
- ✅ **ดูทั้งหมด** - ตรวจสอบการนัดทั้งระบบ
- ✅ **กรองข้อมูล** - แบ่งตามสถานะ
- ✅ **ทำให้เสร็จ** - บันทึกการนัด
- ✅ **ยกเลิก** - ยกเลิกการนัดหากจำเป็น
- ✅ **สถิติ** - ดูสรุปจำนวนการนัด

## 🔄 ขั้นตอนการใช้งาน

### สมมติฐาน: หมอต้องการสร้างการนัดให้ผู้สูงอายุ

```
1. หมอเข้าสู่ระบบ
2. เปิดหน้าผู้สูงอายุ
3. คลิกปุ่ม "สร้างนัดหมาย"
4. กรอกวันเวลา + หมายเหตุ
5. คลิกยืนยัน
6. ผู้สูงอายุได้รับ SMS ยืนยับ
```

### ผู้สูงอายุได้รับการแจ้งเตือน

```
วันที่ 1 (ภายหลังสร้าง):
- SMS ยืนยับการนัด "วันที่ XXXX"

วันที่ X-2 (ล่วงหน้า 3 วัน):
- SMS แจ้งเตือน "การนัดจะมา"

วันที่ X (วันนัด):
- SMS เตือน "วันนี้คือวันนัด"

โดยทั้งนี้จะส่งจากเบอร์ที่ผู้สูงอายุลงไว้
```

### ผู้สูงอายุต้องการเลื่อนวัน

```
1. ผู้สูงอายุเข้าหน้าการนัด
2. คลิกที่การนัดที่ต้องการเลื่อน
3. คลิก "เลื่อนวันการนัด"
4. เลือกวันใหม่ + เหตุผล
5. คลิกยืนยัน
6. หมอและผู้สูงอายุได้รับ SMS แจ้งเลื่อน
```

### หมอต้องการเลื่อนวันเพราะไม่ว่าง

```
1. หมอเข้าหน้าการนัด
2. คลิกการนัดที่ต้องการเลื่อน
3. คลิก "เลื่อนวันการนัด"
4. เลือก "หมอไม่ว่าง" แล้วเลือกวันใหม่
5. ส่งแล้วผู้สูงอายุได้รับ SMS เปลี่ยนวัด
```

### สิ้นสุดการนัด

```
1. หมอเข้าหน้าการนัด
2. คลิก "ทำให้เสร็จ"
3. ข้อมูลนัดย้ายไปแฟ้มประวัติ
4. ผู้สูงอายุสามารถดูในประวัติการนัด
```

## 💾 ข้อมูลที่เก็บ

### ตารางหลัก: Appointment
```
- id: String (PK)
- elderlyId: String (FK)
- doctorId: String (FK)
- appointmentDate: DateTime
- notes: String
- status: SCHEDULED | RESCHEDULED | COMPLETED | CANCELLED
- reminderSentAt: DateTime (เมื่อส่ง SMS แจ้งเตือน)
- appointmentCompletedAt: DateTime (เมื่อทำให้เสร็จ)
```

### ตารางประวัติ: AppointmentHistory
```
เก็บข้อมูลการนัดหลังจากเสร็จแล้ว:
- appointmentDate
- notes
- doctorName
- completedAt
```

### ตารางเลื่อน: AppointmentReschedule
```
เก็บประวัติการเลื่อน:
- originalDate (วันเดิม)
- newDate (วันใหม่)
- reason (เหตุผล)
- rescheduledBy (ใครเลื่อน)
```

## 🚀 การตั้งค่าและปรับใช้

### 1. Database Migration
```bash
npm run db:generate  # สร้าง Prisma Client
npm run db:push     # สลับเสร็จสิ้นอักษร
```

### 2. SMS Configuration (Optional)
```bash
npm install twilio
```

อัปเดต `.env.local`:
```
TWILIO_ACCOUNT_SID=your_sid
TWILIO_AUTH_TOKEN=your_token
TWILIO_PHONE_NUMBER=+1234567890
```

### 3. Cron Jobs Setup
ใช้ Vercel Cron หรือบริการอื่น:
```
POST /api/appointments/cron?type=reminder  (ทุกวันเช้า)
POST /api/appointments/cron?type=today     (ทุกเช้า)
```

## ⚠️ ข้อสำคัญ

1. **ให้ใส่เบอร์โทรศัพท์** - ผู้สูงอายุต้องลงทะเบียนเบอร์โทรเพื่อได้รับ SMS
2. **เตรียม Twilio** - หากต้องการ SMS จริง ต้องตั้งค่า Twilio
3. **ตั้ง Cron Job** - ต้องตั้ง cron เพื่อส่ง SMS อัตโนมัติ
4. **Timezone** - ระบบใช้ timezone ท้องถิ่นโดยอัตโนมัติ

## 🔧 Troubleshooting

### ไม่ได้รับ SMS
- ✓ ตรวจสอบเบอร์โทร
- ✓ ตรวจสอบการตั้งค่า Twilio
- ✓ ตรวจสอบ Cron Job

### ข้อมูลถูกแสดงแล้ว
- ✓ ลบ cache browser
- ✓ รีเฟรชหน้า
- ✓ ตรวจสอบ database connection

### Error ในการสร้างนัด
- ✓ ตรวจสอบวันที่เลือก (ต้องเป็นอนาคต)
- ✓ ตรวจสอบว่าผู้สูงอายุมีอยู่
- ✓ ดูข้อความ error ที่ screen
